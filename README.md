# Fundcatcher

# 使用说明

  在导入参数标签页设定跟踪基金的数目，设定后右侧的表格将提供相应的行。
  在右侧表格输入基金编号及本金，提交。所有设定将由程序保存至setup文件。
  每次启动程序后，界面初始化__init__函数将自动读取setup文件，并进行抓取。
